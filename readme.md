## Features

* Save notes for later use
* Simple and clean UI
* Responsive design (basic)

## Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Flask 
* Database: SQLite

This project was made for learning and practice. It follows a tutorial but includes my own understanding and minor adjustments.

## Getting Started

1. Clone the repository

